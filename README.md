# Superstore Sales Performance Dashboard

## Overview
An exploratory data analysis and interactive Power BI dashboard built on the US Superstore dataset. This project analyses sales, profit, discounts and shipping data to identify business trends and actionable insights.

## Tools Used
- Python (Pandas, Matplotlib) — data cleaning and EDA
- Power BI — interactive dashboard
- GitHub — version control and documentation

## Files
| File | Description |
|---|---|
| `Sales Performance Dashboard.pbix` | Power BI dashboard file |
| `superstore_clean.csv` | Cleaned dataset |
| `Superstore_EDA.ipynb` | Python EDA notebook |

## Key Insights
- Technology is the highest revenue category at $0.84M
- The West region is the most profitable at $108K
- Sales peak in November and December
- Furniture has high sales but very low profit margin
- Heavy discounting negatively impacts profit

## Recommendations
- Reduce discounts on low-margin Furniture products
- Invest more in Technology which has the best profit balance
- Focus growth efforts in West and East regions
- Plan inventory and campaigns ahead of Q4 sales spike
